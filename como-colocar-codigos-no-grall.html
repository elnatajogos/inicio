COMO COLOCAR CODIGOS NO GRALL



No jogo Graal Online Classic existem lojas de roupas, acessórios, chapéus e até botas. Mas também há jeitos de ter escudos, espadas e um perfil diferente e legal.
Para isso existem códigos de PERFIL e de CHAT. Veja abaixo.
Obs: Se seu navegador traduzir o blog escolha nunca traduzir esse site, se não algumas palavras podem mudar e os códigos ficaram errados.

Códigos de perfil

Para mudar a cor de fundo do perfil escreva no STATUS:
<body bgcolor=?> (substitua o ? pelo nome dar cor que você quer no seu perfil em inglês)

Para colocar uma imagem do seu dispositivo como fundo do perfil escreva no STATUS isso:
<body background=?> (troque o ? pelo nome da imagem)

Para mudar a fonte do texto do seu status escreva nele isso:
<font face=?> (substitua o ? pelo nome da fonte escolhida).

Para deixar a fonte do texto do seu status em negrito escreva nele isso:
<b>

Para deixar itálico escreva isso:
<p>

Para deixar sublinhado escreva isso:
<u>

Para cortar as palavras no meio escreva isso:
<slash>

Para fazer as letras piscando escreva:
<blink>

Para aumentar ou diminuir o tamanho da fonte da letra escreva:
<font size=?> (troque o ? pelo número do tamanho desejado desejado)

Para trocar a cor da letra escreva:
<font color=?> (troque o ? pelo nome da cor escolhida em inglês)

Para pular uma linha escreva:
<br>

Para deixar o texto no centro escreva:
<center>

Para inserir uma imagem do seu dispositivo no status escreva:
<img src=?> (troque o ? pelo nome da imagem)

Códigos de Chat

Os códigos de CHAT podem modificar a sua espada ou seu escudo.


Para mudar o escudo escrava isso no CHAT:

setshield 765wings_shield.png (esse código é de asas brancas, você pode trocar o texto em vermelho por outro código que pode ser encontrado pesquisando no Google)

Para mudar a espada escreva no CHAT isso:

setsword iganinja-sword.gif (esse código é de uma espada com um símbolo no meio, você pode trocar o texto em vermelho por outro código que pode ser encontrado pesquisando no Google)
